Explicación

El codigo mostrado en la sección mkdev crea un archivo llamado "archivo1", a este archivo se le inserta una cadena de caracteres utilizandon redirecionamientos de una de las salidas estandard, por lo cual el contenido que el archivo contendra al ejecutar el conjunto de comandos son las siguiente cadena de caracteres.
